# CSL 期末複習筆記

NTU CSIE **CSL (Creative Studio Lab)** 課程期末複習筆記，涵蓋 6 個章節：

1. Dynamics（動力學、PID）
2. Early HCI（人機互動史）
3. Mechanisms, Friction, Transmission（機構、摩擦、傳動）
4. Trusses & Springs（桁架與彈簧）
5. Small & Large Actuation（致動器：壓電 / SMA / 液壓氣動 …）
6. Lasercutter Connections（雷切接合）

筆記是**單一檔案** `CSL_期末複習.html`，內建可收合章節、深入解析面板、自我測驗、SVG 圖解與 KaTeX 數學式。

## 使用方式

1. **Clone 或 pull 這個 repo：**

   ```bash
   git clone <this-repo-url>
   # 之後更新
   git pull
   ```

2. **直接用瀏覽器打開 `CSL_期末複習.html`** —— 雙擊檔案，或把它拖進瀏覽器視窗即可。不需要任何 build、不用裝套件、不用開 server。

> ⚠ **需要連網**：數學式 (KaTeX) 與 Markdown 渲染 (marked.js) 走 CDN，第一次開請保持連線。

## 頂部按鈕

| 按鈕 | 功能 |
|---|---|
| 展開全部 | 打開所有章節 |
| 收合全部 | 關閉所有章節 |
| 顯示/隱藏答案 | 切換所有自我測驗答案 |
| 深入解析 | 切換所有深入解析面板 |
